# OSI Model

> Estimated Reading Time: 10 minutes
> Difficulty: Beginner
> Prerequisites: Introduction to Networking

## Introduction

The OSI Model is a conceptual framework that explains how data moves between devices. Although real networks use the TCP/IP model, the OSI model is widely used for learning and troubleshooting.

## What You'll Learn

- Seven OSI layers
- Layer responsibilities
- Example protocols
- Troubleshooting by layer

## Why It Matters

When someone says "It's a Layer 3 problem," they're referring to the OSI model.

## The Seven Layers

| Layer | Name | Example |
|---|---|---|
|7|Application|HTTP, DNS|
|6|Presentation|TLS, Encryption|
|5|Session|Session Management|
|4|Transport|TCP, UDP|
|3|Network|IP|
|2|Data Link|Ethernet|
|1|Physical|Cable, Fiber|

## Real-World Example

Opening a website involves every OSI layer—from the physical cable to the browser displaying the page.

## Cybersecurity Context

Analysts use the OSI model to isolate problems, understand attacks, and identify where security controls operate.

## Interview Questions

- How many layers are in the OSI model?
- Which layer uses IP?
- Which layer uses TCP?

## Summary

The OSI model provides a structured way to understand network communication.

# IPv6

> Estimated Reading Time: 8–10 minutes
> Difficulty: Beginner
> Prerequisites: IPv4 Addressing

## Introduction

IPv6 was developed to overcome the limited address space of IPv4. It provides a vastly larger pool of addresses and introduces improvements for modern networking.

---

## What You'll Learn

- Why IPv6 exists
- IPv6 address format
- Link-local vs Global addresses
- Advantages over IPv4

---

## Why It Matters

The Internet continues to grow. IPv6 ensures enough addresses for future devices while supporting modern network design.

---

## Main Concept

Example IPv6 address:

2001:db8:85a3::8a2e:370:7334

Unlike IPv4, IPv6 uses 128-bit addresses.

## Real-World Example

Many home networks today run dual-stack, meaning devices use both IPv4 and IPv6 simultaneously.

---

## Cybersecurity Context

Security professionals must recognize IPv6 traffic because it may bypass security controls designed only for IPv4.

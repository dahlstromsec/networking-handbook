# ARP

> Estimated Reading Time: 10–15 minutes

## What You'll Learn
- Purpose of ARP
- ARP Requests and Replies
- ARP Cache
- ARP Spoofing

## Key Concepts

ARP maps an IPv4 address to a MAC address on a local network.

## Useful Commands

```bash
arp -a
ip neigh
```

## Cybersecurity Context

ARP spoofing can enable man-in-the-middle attacks by poisoning ARP caches.

## Summary

ARP enables devices on the same network to discover each other's MAC addresses.

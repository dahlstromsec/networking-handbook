# Network Security

## What You'll Learn
- Firewalls
- IDS
- IPS
- DMZ
- Zero Trust
- Network Segmentation

## Key Concepts

- IDS detects attacks.
- IPS detects and blocks attacks.
- DMZ isolates public-facing services.
- Zero Trust assumes no device is trusted by default.

## Cybersecurity Context

Layered network security reduces risk and limits attacker movement.

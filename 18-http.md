# HTTP

> Estimated Reading Time: 8–10 minutes
> Difficulty: Beginner
> Prerequisites: TCP

## Introduction

Hypertext Transfer Protocol (HTTP) is the protocol used to transfer webpages and other web resources.

---

## What You'll Learn

- Client-server communication
- HTTP requests and responses
- Common methods
- Status codes

---

## Why It Matters

Every time you browse a website, your browser communicates using HTTP or HTTPS.

---

Browser

↓

↓

- GET

Common Status Codes

- 200 OK

- 404 Not Found
- 500 Internal Server Error

HTTP traffic is inspected during web application testing, proxy analysis and incident response.

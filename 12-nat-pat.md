# NAT & PAT

## What You'll Learn
- Network Address Translation (NAT)
- Port Address Translation (PAT)
- Public vs Private IPs

## Key Concepts

NAT translates private IP addresses to public addresses.
PAT allows multiple devices to share a single public IP by using different ports.

## Cybersecurity Context

NAT hides internal addressing but should not be considered a security control on its own.

## Summary

NAT and PAT conserve IPv4 addresses and enable internet connectivity for private networks.

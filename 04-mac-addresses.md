# MAC Addresses

> Estimated Reading Time: 8–10 minutes
> Difficulty: Beginner
> Prerequisites: Ethernet

## Introduction

Every network interface has a unique Media Access Control (MAC) address used for communication on a local network.

---

## What You'll Learn

- What a MAC address is
- MAC address format
- MAC vs IP addresses
- How switches use MAC addresses

---

## Why It Matters

Devices must know each other's MAC addresses before they can exchange Ethernet frames on the same network.

---

## Main Concept

Example MAC address:

00:1A:2B:3C:4D:5E

Think of it as a device's local hardware identifier.

---

## MAC vs IP

| MAC Address | IP Address |
|-------------|------------|
| Used inside a LAN | Used between networks |
| Layer 2 | Layer 3 |
| Hardware identifier | Logical address |

---

## Cybersecurity Context

MAC addresses appear in packet captures and can be spoofed by attackers to impersonate devices.

---

---

## Summary

MAC addresses identify devices on local networks and enable Ethernet communication.
